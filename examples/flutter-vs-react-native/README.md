## 实验说明

本实验采用 PoC 驱动模式，不搞大而全的纸面对比。关键在于：

1. **列出你最纠结的关键假设** — 比如"Flutter 性能更好，但 Dart 人才难招"
2. **用 PoC 验证它** — 写一个最小 Demo，只回答这一个问题
3. **验证通过则推进，不通过则排除** — 每条假设的结论就是决策拼图的一块
